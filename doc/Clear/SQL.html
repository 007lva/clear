module Clear::SQL

Overview

Clear::SQL

Clear is made like an onion:

+

On the bottom stack, Clear offer SQL query building. Theses features are then used by uppermost parts of the engine.

The SQL module provide simple API to generate delete, insert, select and update methods.

Each requests can be duplicated then modified and executed.

Each request object is mutable. Therefor, to update and store a request, you must use manually the dup method.
